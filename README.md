# Python Task Manager (CLI) 🚀

This is my **very first Python project**! It's a clean and functional Command Line Interface (CLI) application designed to help users manage their daily tasks efficiently.

I built this project to practice fundamental programming concepts and to understand the workflow of a real-world Python application.

## 🛠️ Features

- **Add Tasks**: Quickly add new items to your to-do list.
- **View List**: A formatted view of all your tasks with completion status (✓).
- **Update Tasks**: Edit the name of existing tasks.
- **Complete Tasks**: Mark tasks as done with built-in validation.
- **Smart Deletion**: Remove all completed tasks at once using list comprehension.
- **Error Handling**: Robust input validation to prevent crashes if a user enters invalid data (like text instead of numbers).

## 🧪 Concepts Applied

While building this, I learned and applied:
* Variable manipulation and f-strings.
* Control flow (`if/else`, `while` loops).
* Data structures (Lists and Dictionaries).
* Exception handling (`try/except` blocks).
* Git version control and repository management.

## 🚀 How to Run

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/PedroCarvalho711/python-task-manager.git](https://github.com/PedroCarvalho711/python-task-manager.git)
    ```
2.  **Navigate to the folder**:
    ```bash
    cd python-task-manager
    ```
3.  **Run the application**:
    ```bash
    python manager.py
    ```
